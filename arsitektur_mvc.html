Arsitektur Sistem MVC

Dashboard Monitoring dan Rekomendasi Hama — Skripsi Development Architecture


🏗️ MVC Pattern ⚡ Node.js + Express ⚛️ React.js (Vite) 🍃 MongoDB 📡 MQTT Protocol

📖 Penjelasan Komponen MVC — Untuk Bab Metodologi Skripsi

Layer Peran MVC File / Komponen Fungsi Utama
VIEW Tampilan Pengguna Dashboard.jsx, Rekomendasi.jsx, DeteksiHariIni.jsx, RiwayatGrafik.jsx, Login.jsx, dst. Menampilkan data ke pengguna. Mengirim HTTP Request ke Backend via Axios/Fetch.
MIDDLEWARE Penjaga Akses auth.middleware.js, validation.middleware.js, error.middleware.js Validasi JWT Token, validasi input request, dan penanganan error global.
ROUTES Pengarah Request auth.routes.js, detections.routes.js, analysis.routes.js, notification.routes.js Memetakan endpoint HTTP (GET/POST/DELETE) ke fungsi Controller yang sesuai.
CONTROLLER Pengontrol MVC auth.controller.js, detections.controller.js, analysis.controller.js, notification.controller.js Menerima request, memanggil Service, dan mengembalikan JSON response ke View.
SERVICE Business Logic auth.service.js, detections.service.js, analysis.service.js, notification.service.js Mengelola logika bisnis: kalkulasi analisis, generate rekomendasi, hashing password.
REPOSITORY Data Access detection.repository.js, notification.repository.js, user.repository.js Abstraksi query database. Service tidak langsung akses DB, melalui Repository.
MODEL Model MVC detection.model.js, hama.model.js, user.model.js, notification.model.js, recommendation.model.js, rule.model.js Mendefinisikan schema data MongoDB menggunakan Mongoose ODM.
IoT Input Data Sensor mqttHandler.js, Sensor PIR, Kamera YOLO Menerima data deteksi hama secara real-time melalui protokol MQTT dari perangkat IoT.
